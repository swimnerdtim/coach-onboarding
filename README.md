# Swimnerd AI - Coach Onboarding Prototype

Interactive 11-screen onboarding flow for Swimnerd AI mobile app (coach version).

## Live Demo
🔗 https://swimnerdtim.github.io/coach-onboarding/

## Screens

1. **Hero** - "Track your practices with just a picture"
2. **Basic Profile** - Name, DOB, Gender
3. **Number of Teams** - How many teams do you coach?
4. **Primary Team Setup** - Team name, type, level, size, pool
5. **Season Structure** - Season model, current phase, championship date
6. **Training Philosophy** - Coaching approach checkboxes + free text
7. **Practice Structure** - Duration, warm-up/main/cool-down percentages
8. **Go-to Sets** - Favorite set types
9. **Intervals & Pacing** - Clock vs rest intervals, pacing communication
10. **Goals for AI** - What coaches want from the AI
11. **Ready!** - Success screen with next steps

## Features

- ✅ Mobile-first responsive design
- ✅ Progress bar
- ✅ Smooth screen transitions
- ✅ Form validation ready
- ✅ Clean Swimnerd blue branding
- ✅ Interactive elements (checkboxes, radio buttons, team count selector)
- ✅ Data collection for AI personalization

## Data Captured

All data needed to personalize AI-generated practices:
- Coach profile (name, credentials)
- Team context (level, size, pool)
- Season structure (periodization model, current phase)
- Training philosophy (volume vs intensity, approach)
- Practice preferences (duration, structure, set types)
- Pacing/interval methods
- Goals for using AI

## Tech Stack

- Pure HTML/CSS/JavaScript (no dependencies)
- Deployable to any static host
- Mobile-optimized
